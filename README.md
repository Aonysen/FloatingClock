# FloatingClock

[简体中文](README.zh-CN.md)

A lightweight, portable floating clock for Windows 11 x64. It stays above
games and other windows so the current time is always easy to check.

![FloatingClock settings](docs/settings.png)

## Features

- Always-on-top floating clock
- Drag positioning and position memory
- Font family, size, weight, text color, and background color
- Window opacity and multiple time formats
- Position lock and optional mouse click-through
- Rounded background and Windows startup option
- System tray controls and `Ctrl+Alt+T` show/hide shortcut
- Resizable Windows 11-style settings window
- Per-Monitor V2 high-DPI support
- Portable single-file executable

## Compatibility

- Windows 11 x64
- Borderless-windowed or windowed game mode is recommended
- Exclusive fullscreen games may cover normal desktop overlays

## Build from source

No Visual Studio installation is required on a standard Windows 11 system.
The build uses the 64-bit .NET Framework compiler included with Windows.

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\build.ps1
```

The executable is written to `dist\FloatingClock.exe`.

## Usage

- Drag the clock to move it.
- Right-click or double-click it to open settings.
- Use the tray icon when mouse click-through is enabled.
- Press `Ctrl+Alt+T` to show or hide the clock.

Settings are stored in `%APPDATA%\FloatingClock\settings.xml`.

## Project structure

- `src/FloatingClock.cs` - application source
- `tools/GenerateIcon.cs` - deterministic original icon generator
- `assets/` - generated PNG and multi-resolution ICO
- `app.manifest` - Windows compatibility and DPI settings
- `build.ps1` - reproducible x64 build
