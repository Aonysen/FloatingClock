# FloatingClock v1.0.0

First public open-source release.

## Highlights

- Portable Windows 11 x64 floating clock
- Customizable font, size, colors, opacity, and time format
- Always-on-top, position lock, click-through, and startup options
- Resizable Windows 11-style settings page
- Per-Monitor V2 high-DPI support
- Original open-source application icon
- Reproducible build without Visual Studio

Download `FloatingClock-Windows11-x64-v1.0.0.zip`, extract it, and run
`FloatingClock.exe`.
